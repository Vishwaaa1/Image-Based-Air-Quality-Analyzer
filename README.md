# 🌍 Image-Based Air Quality Analyzer

A modern AI-powered Flask web application that estimates air quality from environmental images using deep learning. The platform combines an intuitive user experience with real-time predictions, explainable AI visualizations, and interactive analytics to help users better understand pollution levels and environmental conditions.

---

## ✨ Features

### 🎨 Modern & Interactive User Interface

* Elegant glassmorphism-inspired design with gradient aesthetics
* Drag-and-drop image upload with smooth animations
* Responsive layout optimized for desktop and mobile devices
* Dark mode support with persistent user preferences
* Interactive dashboards and visual feedback components

### 🌫️ Air Quality Prediction

* AI-powered air quality estimation from uploaded images
* Instant prediction results with confidence scores
* Real-time analysis and response generation
* Support for multiple air quality categories and pollution levels

### 🔍 Explainable AI (XAI)

* Grad-CAM visualizations for model interpretability
* Highlights image regions influencing predictions
* Before-and-after comparison views for deeper insights
* Improves transparency and trust in AI-generated results

### 📊 Interactive Data Visualization

* Dynamic AQI gauge with color-coded pollution levels
* Visual representation of PM2.5, PM10, and AQI metrics
* Animated charts and statistical summaries
* Historical prediction tracking and trend visualization

---

## 🧠 Machine Learning Integration

* Deep Learning model trained for image-based air quality assessment
* Real-time inference pipeline for rapid predictions
* Grad-CAM explainability framework for visual interpretation
* Probability distribution visualization for model confidence analysis

## 🚀 Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/Dharsan2024/Image-based-air-quality-analyzer.git
cd Image-based-air-quality-analyzer
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

Open your browser and navigate to:

```text
http://127.0.0.1:5000
```
